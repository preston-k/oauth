1. We Use 256-bit Hashing for Passwords

In our system, passwords are secured using SHA-256 hashing. This is a cryptographic hashing algorithm that produces a 256-bit (32-byte) hash value, often rendered as a hexadecimal number with 64 characters. It's a one-way process, meaning that once a password is converted into its hash code, it cannot be reversed back to the original password. This method ensures that even if someone gains access to the hashed password data, it is nearly impossible for them to decipher the actual password. Additionally, the hashing process is performed in a secure, asynchronous manner, utilizing the Web Cryptography API, which provides a strong foundation for security in web applications.

2. We Utilize Anti-Robot Systems to Prevent Robotic Cyber Attacks

Our website utilizes reCAPTCHA to prevent robots from spamming the website. It also prevents user movements that may be rendered as robotic to prevent cyber attacks. Please note that third-party companies such as reCAPTCHA have their own privacy polciies and terms of use. Please review them before continuing on our site.

3. HTTPS Means Secure

Our site uses HTTPS, the secure version of HTTP, ensuring all data exchanged between your browser and our website is encrypted. This means any information you submit, from passwords to personal details, is protected against unauthorized access. HTTPS is a key security feature that guards against eavesdropping and ensures your data remains confidential and secure during transmission.
